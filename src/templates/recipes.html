The Media Suite tools offer the core functionalities needed for performing scholarly research tasks with audio-visual media and contextual collections. These tools have been co-developed with media scholars, oral historians, and other scholars who use audio-visual media intensively. They are also inspired in in prior projects, such as AVResearcherXL, Comerda, DIVE and TROVE. Each tool has been assembled using one or more components.

Collection inspector

With this recipe it is possible to: 1) select collections, 2) inspect their metadata, 3) perform data analytics tasks on their metadata (e.g., detect incomplete data, observe value distributions along date fields), and 4) send them to the other browsing and search recipes.

Exploratory browser

DIVE+ browser is a linked-data, event-based browser, based on the simple event data model, where you can browse and search in an ”infinite” fashion in different heritage collections at the same time. It supports the creation of browsing narratives.

Basic faceted search

Dedicated tool for searching/exploring through a single collection and all available annotation layers for that collection (e.g., automatically-generated, user-generated, archival generated).

Comparative search

Supports comparative (re)search based on multiple queries on the same or different collections.
