 Functionalities

Check out our useful components
In LABO each functionality for the end-user is available in the form of a UI component, check out the components page to see what we have to offer.

 Try out different APIs

Currently we have APIs for search, entity extraction and speech recognition
We're implementing several services that do stuff with our catalogue or can process different kinds of data. Each of these services have APIs which you can try out here. So far we have search, entity extraction and speech recogntion. We plan to have many more APIs in the near future, also exposing services for face recognition and recommendation software.
