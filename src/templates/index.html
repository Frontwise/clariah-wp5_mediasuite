Welcome to the CLARIAH Media Suite


 Recipes

Try one of the predefined recipes, based on existing media studies use cases
Recipes basically consist of a number of functionalities, 'ingredients', with a particular configuration suited for your type of research.

 Functionalities

Visit the component library and try out the available functionalities
Each functionality in the media suite stands on it's own and can be combined with others to form recipes. Recipes can be created by either configuring recipes in your user space or by downloading the component library to be used in your own HTML/Javascript page

 APIs

If you're up for it, use our APIs to build your own functionalities
Currently the CLARIAH infrastructure exposes two APIs: 1) the search API for searching through all of the collections imported from CKAN and 2) the annotation API, which allows you to store and retrieve any annotations made within the CLARIAH media studies infrastructure.
