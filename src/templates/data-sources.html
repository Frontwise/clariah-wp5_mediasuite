The CLARIAH Media Suite provides access to "established" collections from different institutions (e.g., television broadcasts from The Netherlands Institute for Sound and Vision, selected sub-collections from EYE Film Museum, newspapers from the National Library (KB), thematic oral history collections from DANS, among others). All collections (and sub-collections) are registered in a common inventory built in CKAN. CKAN is an open source platform for collection registration, it is a software solution that provides the tools to "streamline publishing, sharing, finding and using data", making it accessible. CKAN aims to making collections all over the world easily discovered and accessed.

In the CLARIAH Media SuiteÂ´s instance of CKAN you can find the metadata about the collections, and the metadata (i.e., the data) of the collections. Most collections are available both in Elasticsearch and RDF formats, and some in either of the two. Access to the media objects (i.e., to the content itself) depends on the policies of each data provider. The data that is "registered" in CKAN is used by the tools in the Media Suite.
