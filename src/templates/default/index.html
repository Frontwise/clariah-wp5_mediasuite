Welcome to LABO


You're currently viewing the default home page of LABO. If you want it to be less boring, please create a folder in:

/templates/custom and pass on the name of the folder in your settings.py in CUSTOM_TEMPLATE_DIR

After that, it's possible to create your own: index.html, contact.html, about.html

For the moment it is not possible to override any of the other pages, but this will be possible in a later version.
